# AMR Portfolio | Full-Stack & AI Specialist

A high-performance, award-winning portfolio website designed with a focus on immersive user experience, technical precision, and modern aesthetics.

## 🚀 Features
- **Interactive Particle Orb**: Real-time canvas-based physics engine with "breathing" animations and scroll-parallax.
- **Bi-Directional Dark Mode**: Seamless theme switching with system-level detection and persistence.
- **Steel-Grade Branding**: Custom "AMR" logo with high-gloss metallic gradients.
- **Responsive Architecture**: Fully optimized for mobile, tablet, and ultra-wide displays.
- **Secure Communication**: Hardened contact system with bot-protection and EmailJS integration.
- **Professional Analytics**: Visualized performance stats and career milestones.

## 🛠️ Tech Stack
- **Frontend**: React 18, TypeScript, Vite
- **Animation**: Framer Motion (motion/react)
- **Styling**: Tailwind CSS (JIT Engine)
- **Icons**: Lucide React
- **Infrastructure**: Environment-driven configuration

## 📦 Deployment
This project is production-ready. 

### Local Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Build for production: `npm run build`

### Environment Configuration
To enable the contact form, create a `.env` file in the root with:
```env
VITE_EMAILJS_SERVICE_ID=your_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_key
```

## 🎨 Design Philosophy
The site follows a **"Human-AI Collaborative"** aesthetic, blending brutalist typography with soft, organic particle movements. Designed to be accessible, fast, and distinctly professional.
